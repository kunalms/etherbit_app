# Etherbit_app

[![N|Solid](https://cdn.shopify.com/s/files/1/2877/5560/files/apple-icon-180x180_32x32.png?v=1516964690)](https://www.etherbit.in/)

Etherbit app is a simple web app to track all the realtime prices of cryptocurrencies

## Installation


Install the dependencies and devDependencies for client and server.

```sh
$ cd etherbit_web
$ npm install
$ npm run client-install
```

For starting server and client concurrently 

```sh
$ npm run dev
```
